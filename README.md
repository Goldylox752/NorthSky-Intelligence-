🚀 NorthSky AI Engine

A high-performance AI-powered metadata & media extraction API built for developers, automation tools, and SaaS products.

💡 SaaS Pitch

NorthSky AI Engine is a scalable API that transforms any URL into structured, usable data in seconds.

It extracts:

Website metadata (title, description, images)
Video metadata (YouTube, TikTok, Instagram, Twitter)
Direct media links using advanced extraction engines
Clean, structured JSON for instant integration

Built for developers who need reliable content intelligence at scale.

⚙️ Tech Stack
Node.js (runtime)
Express.js (API framework)
yt-dlp-wrap (video extraction engine)
metascraper (web metadata parsing)
Winston (logging & monitoring)
express-rate-limit (abuse protection)
Redis (optional scaling layer for production rate limiting)
